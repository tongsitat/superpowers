## Idempotency

All operations check for the `SUPERPOWERS` marker before writing:

- Marker present вҶ’ Skip (already installed)
- File exists, no marker вҶ’ Append superpowers block (preserves existing content)
- File absent вҶ’ Create new file

Safe to re-run the installer at any time.
